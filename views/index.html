Exercise Tracker

User Story

GET user's exercise log: GET /api/users/:_id/logs?[from][&to][&limit]

GET a list of all users from GET /api/users

To retrieve a full exercise log of any user GET /api/users/:_id/logs

You can add 'from', 'to' and 'limit' parameters to a GET /api/users/:_id/logs request to retrieve part of the log of any user

from, to = dates (yyyy-mm-dd); limit = number

Register New User

POST /api/users

Add Exercise

POST /api/users/:_id/exercises
